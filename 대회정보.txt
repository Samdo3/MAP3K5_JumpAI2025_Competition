[주제]
MAP3K5 IC50 활성값 예측 모델 개발


[설명]
PubChem, ChEMBL, CAS 등에서 수집한 실험 기반 화합물 정보를 기반으로 IC50값 예측모델 개발

127종 화합물의 구조 정보를 입력값으로 사용하여 해당 화합물들의 ASK1에 대한 IC50 값을 예측하여 제출

1. 리더보드
평가 산식
A = IC50(nM) 단위의 Normalized RMSE 오차, 예측 정확도 측정


B = pIC50 변환값을 기준으로 한 예측값과 실제값 간의 선형 상관관계의 제곱


Score = 0.4 x (1 - min(A, 1)) + 0.6 x B


Public score : 전체 테스트 데이터 중 사전 샘플링된 40%
Private score : 전체 테스트 데이터 중 나머지 60%

- Substance Name

- Source Type
Type of source material, Journal or Patent

- Document Identifier
Source material identification string, doi, patent number

- SMILES
Chemical structure string,  can also be used as a key to match information in the "Ligand Number Names SMILES" sheet to the "MAP3K5 Ligand IC50s" sheet when Substance Name is blank

- Assay Name
Author given name of the assay used to measure thee IC50

- Measurement Function
Defines the type of target modulation (inhibitor, activator, modulator); EC50s with this field indicating Inhibitor are considered IC50s

- Assay Parameter
Parameter or bioactivity name given to the value being measured, IC50 or EC50 (EC50 with inhibitor designation)

- Display Measurement
Measured IC50 value and units 

- Measurement Prefix (Parsed)
For the measured value this field gives a value modifier where available, equals, less than, greater than

- Single Value (Parsed)
A single measured value (number) for the IC50

- Low-End Value (Parsed)
The lowest measured value in a range of values for an IC50

- High-End Value (Parsed)
The highest measured value in a range of values for an IC50

- Measurement Unit (Parsed)
Unit of measure used for all values

- pX Value
-log(molar concentration of IC50)

- Cell Line
Type of tissue culture cells used for the assay

- Experimental Environment
Whether the assay was carried out in vitro, in vivo, or unknown/not recorded

- Species
Animal species used in assay
